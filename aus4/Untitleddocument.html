FIRST RUN BUTTON AND RUN BUTTON

First Run Button:

User can run the DU/LPA first time by clicking First run button First run button will be displayed under certain conditions for that particular loan.

  • In Encompass tools section when user clicks on AUS tracking, There should not be any entry 
  • Loan Folder should be Prospect or active 
  • Subject property Address should be TBD or Empty.

Run Button

  • User can run the DU/LPA/Both DU & LPA using the application.
  • User should be update all the mandatory fields before run.
  • Run button will be enable in the pipeline under certain conditions for that particular loan
  •  Submission number is greater than 1 in Encompass AUS tracking
  •  Milestones within specified list <File Started,Sent to Setup,Sent to processing,Ready for UW,Submittal,Underwritten, PTD Submittal, Approval>
  •  Loan should be Active Loan folder or in Prospect
  • Subject property Address should be TBD or Empty
